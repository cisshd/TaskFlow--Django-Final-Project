# TaskFlow   
Task Management Platform for Teams

TaskFlow is a full-stack task management web application developed as my **final project for the Django course at Tuwaiq Academy**.  
The platform is designed to help teams and small businesses organize their work efficiently by managing tasks, priorities, and workflows.

---

##  Project Overview

TaskFlow allows users to create, manage, and track tasks in an organized and intuitive way.  
The system focuses on productivity, clarity, and ease of use, making it suitable for collaborative environments and small teams.

This project demonstrates my ability to build a complete web application using **Django** with both backend logic and frontend integration.

---

##  Key Features

- Create new tasks
- Update existing tasks
- Delete tasks
- Set task priorities
- User-friendly interface for task management
- Structured backend with Django best practices
